# User Management System

A Spring Boot backend project for managing users with secure authentication and CRUD operations.  
Designed with layered architecture and integrated with MySQL for persistence.  

---

## Features

- User registration and authentication  
- JWT-based security for APIs  
- Create, Read, Update, Delete (CRUD) operations for users  
- Custom exception handling for error scenarios  
- RESTful API endpoints  

---

## Tech Stack

- **Language:** Java  
- **Frameworks:** Spring Boot, Spring MVC, Spring Security  
- **Database:** MySQL  
- **ORM:** JPA, Hibernate  
- **Tools:** Maven, Postman, Git  
- **Other Concepts:** JWT Authentication, Layered Architecture, Exception Handling
